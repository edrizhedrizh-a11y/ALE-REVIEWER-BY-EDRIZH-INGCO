## Visual rules
- Identify/image-based questions show an unlabeled image in the question card.
- The same image target is reused in the explanation after answering.
- Physical device/equipment definitions show images only after answering.
- Code values, formulas, acronyms, and theory items use concept visuals or no visual.
- Primary image source: Wikipedia / Wikimedia Commons.
- Fallback: Google Images search target only.
